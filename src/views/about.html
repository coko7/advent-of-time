The Advent of Time "Guess what time it is in Stockholm" game takes place from 01/12/2025 to 25/12/2025.

üì∑ The Game

Every day, I will post a picture I have taken in Stockholm.
Your goal is to guess the exact time of the day the picture was taken (HH:mm).
The closer your guess is to the actual time in the picture, the more points you will get.

If the picture was taken at 15:30 and you guessed 16:45 (01:15 diff), you will get 7 points.
Each daily guess will grant you points and the winner will be the person with the most points by December 24th 2022 at 23:59.

‚ö†Ô∏è Important thing to note:

If the actual time of the picture is 01:00 and you guessed 23:00 it will be considered a 21 hours difference (not 2 hours!).
Keep that in mind before submitting your guesses.

Please note that if you do not submit a guess for a day, it will be considered as if you did not participate for this day and you will get 0 points.
So, make sure to guess every single day if you want to have a chance to win.

üöÄ How much Quantum computing power is required to run this site?

This is pure HTML, CSS (and vanilla JavaScript).
No framework, no bullshit. Entirely served by a rusty web server.
You can find all the source code for this webapp here. It's MIT-licensed.
The webapp relies on a home-made and insecure implementation of HTTP 1.1: RTFW-http-rs
For the reverse-proxy, I picked Caddy because it is easy to configure and fast.
Finally, all of this, is being served from a Raspberry Pi, sitting comfortably in a frenchman's apartment. üá´üá∑

üíÄ Can I crash this site?

It should be fairly easy.
Especially given the fact that RTFW-http-rs is most likely (one of) the most insecure HTTP implementation in existence.
I would prefer if you DID NOT try to take down this instance, though.
I can't guarantee I will get the site back up right away and it might ruin the experience for other players.
If you want to experiment, feel free to try and self-host this yourself. It's all open-source, baby.
